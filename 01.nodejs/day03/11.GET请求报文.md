# GET 请求报文：

1. 报文首行
  GET http://localhost:9527/?name=jack&age=18  HTTP/1.1

  请求方式：GET（查） / POST（增） / PUT（改） / DELETE（删） / OPTIONS

  常见的POST请求：Form action="请求地址" method="POST"
  除此以外就是GET

  请求参数：?name=jack&age=18 查询字符串（query string）

  协议名/协议版本 HTTP/1.1

2. 报文头部
3. 空行
4. 报文体
